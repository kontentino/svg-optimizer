# SVG Optimizer
Optimizing svg icon files like [feather-icon](https://github.com/feathericons/feather) package.

## Getting Started

### Prerequisites
- Installed `node` and `npm`

### Install
1. clone repository
2. run `yarn`
4. run `yarn start` or `yarn start:prod`
